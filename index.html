How FlashPatch Works

One exploit triggers a full autonomous response chain — no human in the loop.

01
🔍

Detect

Exploit-detector polls Solana RPC every 2s. Spots anomalous withdrawal pattern on vault program. Broadcasts EXPLOIT_DETECTED + WANT to all patch agents.

→
02
⚔️

Compete

Three patch agents (fast / deep / hybrid) race to submit bids with price + confidence. Detector picks highest-confidence bidder and opens an arbiter-gated escrow on devnet.

→
03
✅

Verify

Sandbox-verifier independently checks the patch: SHA-256 hash, JSON structure, exploit replay simulation, static analysis results. Issues VERIFIED pass or fail.

→
04
🚀

Deploy

Threshold-deployer waits for 2 verified passes, selects best patch, signs a Solana memo proving upgrade authority, broadcasts PATCH_DEPLOYED with on-chain tx sig. Escrow released.

🏗️ Architecture

Wire Protocol

Five FlashPatch message types extend the base CoralOS market protocol — all round-correlated, pure format/parse.

EXPLOIT_DETECTED

Broadcast by exploit-detector. Contains program ID, exploit type, severity, and vault balance at risk.

PATCH_SUBMITTED

Sent by winning patch agent. Contains patchId, instruction, staticAnalysis, confidence, vulnerabilityClass.

SANDBOX_RESULT

Issued by sandbox-verifier. Pass/fail verdict with hash match, structure check, and exploit replay result.

THRESHOLD_STATUS

Broadcast by threshold-deployer after each verified pass. Shows progress toward deployment consensus.

PATCH_DEPLOYED

Final message. Contains on-chain tx sig, elapsed_ms, winning patchId, and upgrade authority proof.

🚀 Get Started

Run It Locally

Four commands. One Docker network. All six agents live within 30 seconds.

1

Clone & Install

git clone https://github.com/Tasfia-17/flashpatch.git
cd flashpatch
npm install
2

Configure Environment

cp .env.example .env
# edit .env: add OPENROUTER_API_KEY
# and SOLANA_PRIVATE_KEY (devnet wallet)
3

Build Agent Images

chmod +x build-flashpatch.sh
./build-flashpatch.sh
# builds exploit-detector, patch-generator,
# sandbox-verifier, threshold-deployer
4

Launch & Watch

docker-compose up    # start CoralOS
npm run flashpatch   # register 6-agent session
npm run flashpatch:web  # open dashboard :3030
