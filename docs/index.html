Architecture

Built on proven rails.

🪸

CoralOS

Agent orchestration, session management, MCP coordination. Six-agent graph declared in JSON, spawned as Docker containers, communicated over shared message bus.

◎

Solana

Arbiter-gated escrow program deployed on devnet. Funds locked on deposit, released only through neutral 3rd signer after verified delivery.

📋

Patch Registry

Pre-audited patch candidates per vulnerability class. Agents compete on price and analysis depth, not on generating Rust from scratch under fire.

🛡️

Policy Engine

Every deposit and release passes enforce(). Spend caps, payout binding, verifier gate. A missing VERIFIED verdict leaves funds in escrow, refundable after deadline.

Get Started

Run it yourself in 5 minutes.

01

Clone and setup

git clone https://github.com/Tasfia-17/flashpatch.git
cd flashpatch
npm install --prefix scripts
node scripts/setup.js

Generates two devnet wallets. Fund the buyer at faucet.solana.com.

02

Configure .env

VENICE_API_KEY=your_key_here
SIMULATE_EXPLOIT=1
SECURITY_BUDGET_SOL=0.05
THRESHOLD=2

Free LLM credits: sign up at venice.ai, redeem code IMPERIAL50.

03

Build and launch

docker compose up -d coral
bash build-flashpatch.sh
npm run flashpatch

Prints the session ID. Exploit fires 5 seconds after startup in simulate mode.

04

Open dashboard

FLASHPATCH_SESSION_ID=<id> \
  npm run flashpatch:feed

npm run flashpatch:web
# opens localhost:3030

Watch all six agents transact live. Three Explorer links at the end.
